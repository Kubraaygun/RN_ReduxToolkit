# User Management Application (Using Redux Toolkit)

This project was prepared to understand the basic usage of Redux Toolkit and to create a user management system. Basic functions such as retrieving user data from API, adding new users and viewing details were implemented in the project.

# Project Structure

## Main Technologies

- React Native: Mobile app development
- Redux Toolkit: State management
- Axios: API requests
- Formik & Yup: Form management and validation

# Features

- State management with Redux Toolkit.
- API integration.
- User management (add, list, view details).
- Form validation with Formik and Yup.

Redux Toolkit, geliştiricilerin karmaşık durum yönetimi yapılarını sadeleştirir ve proje içerisinde daha temiz bir kod yapısı sağlar.
